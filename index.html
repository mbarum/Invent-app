Masuma EA Hub

This is the root of the Masuma EA Hub application monorepo. The project contains separate frontend and backend services designed to work together as a single-page application (SPA).

For Local Development:
Run npm run dev from this root directory. This command starts both the frontend and backend servers concurrently. Access the application at the local address provided by Vite (usually http://localhost:5173).

For Production Deployment (e.g., on CloudPanel):
1. Build the entire application using npm run build.
2. Configure your server to run the Node.js application from its entry point: backend/dist/server.js.
3. The backend server is configured to serve the static frontend files. No separate web server for the frontend is needed.
